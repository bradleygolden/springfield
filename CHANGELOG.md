# Changelog

All notable changes to Springfield will be documented in this file.

## [1.0.0] - 2025-11-05

### Added
- Initial release of Springfield plugin
- Multi-phase autonomous workflow orchestration
- Lisa command for codebase research
- Mayor Quimby command for complexity assessment
- Frink command for implementation planning
- Homer command for autonomous implementation
- Comic Book Guy command for QA validation
- Ralph command and skill for full workflow orchestration
- Debate refinement loop for complex tasks
- Ralph implementation loop for iterative development
- Language-agnostic quality checks
- Session-based workflow management

### Features
- Semantic skill invocation from slash commands
- Autonomous Ralph skill activation
- Helper scripts for debate and implementation loops
- Context-efficient workflow design
- Adaptive feedback loop with QA kickbacks
- Progressive disclosure of documentation
